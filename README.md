## 使用说明

1. 所有文件按类别分类存储在相应目录下
2. 电子书籍统一存放在 `books/` 目录
3. 财务报表和其他数据文件存放在 `data/` 目录
4. 建议定期更新此索引文件以保持文档的可查找性

## 维护建议

1. 添加新文件时及时更新本索引
2. 保持文件命名的规范性
3. 定期备份重要文件
4. 建议为文件添加简要描述或关键字标签
